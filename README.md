# Sentiment Analysis

A machine learning project for classifying text sentiment as Positive, Negative, or Neutral using NLP techniques.

## 📌 Features
- Text preprocessing
- Sentiment prediction
- Machine Learning model
- Easy to use

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK

## 📂 Project Structure
sentiment-analysis/
│── README.md
│── sentiment_analysis.ipynb
│── dataset.csv

## 📊 Model Used
- Naive Bayes

## 🚀 How to Run
1. Open notebook in Google Colab
2. Run all cells
3. Enter text for prediction

## 📈 Output
Predicts whether input text is Positive, Negative, or Neutral.
